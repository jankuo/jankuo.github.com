花开春不暖，春到雨不停

发表于 2017-06-20 | 分类于 情感 |

上午又调休了，原因是为了避免在今天的周会上不满某些作风和做法，一怒喷领导（话说毕业三年，这个破脾气是越来越盛了）。可下午上班的时候就后悔了，哇塞，今天好多人没去开会啊，包括那个一直在忽悠我的领导，亏了3个小时，嘎嘎。

下午呢？又开始无聊到家了，看了几个算法题，无趣，找同事《盗墓笔记》接龙，无聊。研究新技术，无意。苍天啊，救救我吧，请 赐予一些书给我看吧，比如盗墓类的小说；比如《失控》之类的经典。或者把我的记忆抽走吧，这样又有很多书可以让我看了，具的过目不忘真可怜，别人《盗墓笔记》能看第二遍，第三遍，我一遍看完就能背出来了。

晚上更悲剧，发现中午的碗还没洗。碗啊，筷子啊，高压锅啊…..吃的时候没感觉，洗的时候那个多啊。
也不知道是厨房本来就很湿，还是今天倒霉。来回几次，把厨房搞的像刚下雨似地。

那个破水龙头，还跟我闹脾气，动不动就会漏水，滴啊滴的，受不了。

偶然发现，厨房里还有香菇 （上次炖鸡肉吃的附加品），于是赶紧把它拿出来，蒸了，可怜的5，等下又要洗碗了。

连续的阴雨天，厨房里的竹子厨具里能生那个 斑（不是逗逗，也不是雀斑）的，看起来都班痕累累了，气死我了。。。。。。。。。。。。

如果你也是一个对过去念念不忘的人，建议你听听《我们失去的家园2》，很感伤，尤其是像5这样离家的孩子，在静静的夜，听着这些歌曲，眼泪那个哗啦啦的流啊（以前会，现在坚强了），当然，你也得跟我一样，第二天早上怀着“今天努力做好，明天就有希望”的心情。不然N年（2^10)后，佛祖会怪我的。

苍天啊，掉个500W给我，我回家打造 若亚方舟 逃避2012去。

话说过两天啊鹏要来福州了，明天又要交房租了，我可怜的钱包啊。

过几天把《注册会计师》的书买来看看，既然决定要全能了，就更全能点吧，顺便看看能不能当小说看，可以消解下无聊的时光。

一念之间

发表于 2017-06-20 | 分类于 情感 |

自从迷上了 《霹雳布袋戏》后，人就像着了魔一样，每周5晚上一定要看完两集才睡，每周一晚上一定要看完预告才安心。现在的《天竞鏖锋》，里面有个场景叫一念之间，里面的主人叫《圣魔之仆》。虽然不喜欢这个人不人，动物不动物，妖怪不妖怪的…但对一念之间还是非常有爱，因为：

曾经他的女儿，在父爱与情爱之间，在正义与父爱之间，徘徊良久，最后选择与父亲同归于尽；
曾经最爱的玉辞心姐姐，在仇恨与母子之情之间徘徊，最后选择了舍己救爱子；
曾经最爱的双胞胎，就在这个地方，开始了他们为圣为魔的人生，虽然现在圣以杀止杀，魔以统止杀，在这条被武林唾弃的道路上，却在为天下苍生造福；
曾经的这里，帝如来，鬼如来。一身两面像；在武林圣为魔，魔为圣的道路上，坚持以己到，正武林。

今天，在欲走还留的徘徊里，我选择了离开。

离开了这家，我曾经很尊敬的公司，曾经就有人劝导我不要进的公司，曾经憧憬着制度和流程都很完善，憧憬着能够学到很多技术，憧憬着能够深造某门技术，却在一次又一次的打击中，一次一次愤怒，一次一次伤心，一次一次心灰意冷。

离开了这个，我爱不能弃不能的项目。

也只有时至今日，才渐渐明白，为什么那些离开网龙，又非常怀缅网龙的人，那种恨恨不舍的心态。

唯一的欣慰，就是，离开的时候又认识了某些女生。

相逢足矣，何必相识，今日是同事，离开了就是行路人。

网龙的文化，DJ所创导的网龙体制，是那么的优美。

却在高层内部某些老死不相往来；某些高层压制下面的不能晋升，某些高层压制绩效考核的种种弊端下，渐渐失去了原有的风采。

一个新的项目的成立，总有那么几个领导，想分一杯羹，有些甚至纯粹就是为了分羹而进来的。做事的时候，这个不会，那个不会，问这个不懂，那个你自己想办法，这个你延迟下，那个你克服下。

也深深知道了在网龙这些大公司，体制制度都很完善的前提下，要翻一个身，是多么的困难，要诞生一个新的赢利点，是多么的困难。
所以，才渐渐不那么憎恨腾讯，百度和阿里的C2C.毕竟，人家想抄袭还是需要很多步骤的。

走吧。我还年轻；我还没结婚，可以承受离开所带来的压力。

一念之间。我的心已经远去

九中之殇

发表于 2017-06-20 | 分类于 回忆的咖啡 |

阿鹏的到来，的确让我欣喜，于是悲剧的又请了1个小时的假，这周假请的我都不好意思了！

晚餐一时兴起，带着阿鹏去五四路的黄鹤楼去吃大餐了，不知道这算不算堕落。久违的同学，久违的朋友，6年半后再相聚，任何话都闲多，只有一个字，吃。可惜的是除了招牌菜 “泡椒田鸡”外，其他的菜只能是马马虎虎的了。连最悲剧的酒也只有低等的 长城干红。我擦，可惜这次楷悲剧了，被叫去陪大区经理等了，无缘据说是福州最好吃的泡椒田鸡。

席间最多的话题不是工作，不是近况，竟然是女人，是不是大家都到了岁数，都要结婚了。谈到女朋友，当然少不了那个人了，具体的，知道的不说也知道，不知道的也不需要知道，既然快成为历史了，知道了又能改变什么呢？

晚上三个人，几片猪什么肉，什么什么鸡爪。一箱啤酒。XXX，三个人又畅饮了起来。可惜的是,由于我刻意限制，今晚啤酒喝的真的很少

明天，福州的高中同学聚会。看来是九中之殇的重提与破冰。尘封多年的记忆，终于要在见到越来越多的高中同学后，渐渐被大脑挖掘。我还是过去的那个我吗？如果是，那这么多年不愿意正视高考失利的我，不愿回忆那三年生活的我，还能捡回以前的性情吗？如果不是，为什么每次想到那三年，就会心潮澎湃。

我知道，有些话题，即使见到再多了人，也不会有人提及，但阿鹏和素芳是个例外。但这个例外的人来了，还真提及了。

一生中，两个地方始终无法忘怀，一个是外婆的老家，另一个是九中。外婆的老家，基本已经作古，留下的只有伤心与遗憾，这份遗憾，越来越让我看重亲情，珍惜目前身边所有得到的东西，在我最想放弃的时刻，想起曾经的放弃，造就了我多少的遗恨，心就会一直坚持下去。而九中，这个感情复杂的地方，至今为止，我都不知道垓怀着什么样的感情来承受过去与未来。

在事业越来越接近成功的时刻，对九中高考失利的复杂感情应该是越来越清晰的，毕竟过去的这么多选择，我还是错到了好处，虽然现在不能算非常成功，但以我的才能，我相信终究会有那么一天的到来。

而另外一项，也只有等到我结婚的时候，或者说结婚的N多年后。才会有最终的解法。我期待那一天快点降临。毕竟这道坎，迈过去了就过去了，没有迈过去，那么永远都是道坎。

上天有门，入地有道，前方广阔，后方平坦，奈何没有我心之出路

心结所在，何处可解？

然每次抬头，就会发现眼前的不是自行车，就是行人，甚至是小汽车。

那种感觉似在表明自己，即使自己不看前方，也不会撞山，不会落水，不会碰行人

俯瞰山底，不过是一批又一批小如蚂蚁的登山者，似曾相识的情境，勾起遗憾的往事

平视远方，依稀的几座高山，遮住了迷离的睡眼

题目：不用第三方变量，交换 a 与 b 的值

我的回答：

1.

a = a+b  
b = a-b  
a = a-b

2.

a = a\^b  
b = a\^b  
a = a\^b

本以为这样的回答应该可以了，虽然这两个答案的极限至于 a,b 必须是整数，而且 2 方案中 a 不能大于 2^8 。但题目也没有说明a，b是什么数。

但面试官给我的答案是：

不够快，啥？不够快？然后说PHP有个函数可以实现

我顿时无话可说了，像我这样看api还挺勤劳的人来说，对这个函数只有印像，何况一般人，如果把这个函数用于实际代码，不知道有多少人要查查api，然后破口大骂写代码的人祖宗十八代。何况，从性能上来说，那个函数的性能还不一定比我的快。

我说，如果用那个函数，写的人没什么，但修改代码的人估计每次都要翻api！

人家说：你没看api吗？有函数就可以用，为什么要自己造车。

后来听hr人事说，那个招聘官是php界的大牛？大牛，还不如说是php的极客。

除了这题，其他都没意见，如果我的答案是正确的，笔试题都正确了，你拿什么来压低我的工资。

于是果断的把期望工资提高了一个等级，不是我要提高，而是在这样的领导下，我都不知道是教条点呢还是灵活点，XXX，工资没达到期望，果断不去中。
