想说爱你，不想开口，想说离开，难以割舍

发表于 2017-06-20 | 分类于 情感 |

都说大公司标准，规范。Y的，不规范的时候不规范到家了。

一个垃圾破网站，都做了5个月了，还是一无所是。我们这些当兵的，累死累活天天加班，天天把某个需求，改了又改。总共就那么几个破页面，做的什么都不像

项目的老大，只会最后瞄一眼，平时做成什么样，不管不问

项目的二老大，也是在老大问的时候，才会关心这个关心那个，还负责营销的，我X，连页面怎么设计也不管，就Yy的在最后的时候说这个不行，那个不行。

项目的三老大，就是一个忽悠人，其他能力，都他吗的一点都看不出来

项目的四老大，被我损的，损的我自己都不好意思了

还有需求方，一个破电商网站，让一个从事传统电话营销的来做需求分析，业务主导型。XX，什么概念。这个还叫电商吗？干脆叫山寨阿里巴巴的了

星期一到星期五，闲的时候从早上喝绿茶都一直喝到下班。吃饭才吃5元，绿茶喝了5瓶

平时请个假调休，比登天还难，这也就罢了

一到星期5晚上或者双休的时候，就他吗的事多，每天都有被加班的可能性出现

还整天说你们要爱这个项目，这种爱说的出口吗？

想离开，但这个是我毕业以来，同事最多的一个公司，想离开，建立起来的同事感情。很难割舍的。

明天又要去加班了。他YY的真难受
