你拿什么来跟我谈工资

发表于 2017-06-20 | 分类于 情感 |

题目：不用第三方变量，交换 a 与 b 的值

我的回答：

1.

a = a+b  
b = a-b  
a = a-b

2.

a = a\^b  
b = a\^b  
a = a\^b

本以为这样的回答应该可以了，虽然这两个答案的极限至于 a,b 必须是整数，而且 2 方案中 a 不能大于 2^8 。但题目也没有说明a，b是什么数。

但面试官给我的答案是：

不够快，啥？不够快？然后说PHP有个函数可以实现

我顿时无话可说了，像我这样看api还挺勤劳的人来说，对这个函数只有印像，何况一般人，如果把这个函数用于实际代码，不知道有多少人要查查api，然后破口大骂写代码的人祖宗十八代。何况，从性能上来说，那个函数的性能还不一定比我的快。

我说，如果用那个函数，写的人没什么，但修改代码的人估计每次都要翻api！

人家说：你没看api吗？有函数就可以用，为什么要自己造车。

后来听hr人事说，那个招聘官是php界的大牛？大牛，还不如说是php的极客。

除了这题，其他都没意见，如果我的答案是正确的，笔试题都正确了，你拿什么来压低我的工资。

于是果断的把期望工资提高了一个等级，不是我要提高，而是在这样的领导下，我都不知道是教条点呢还是灵活点，XXX，工资没达到期望，果断不去中。
