最近的一些烦心事

发表于 2017-06-20 | 分类于 心情 |

过去一周，过得很纠结，好几个夜晚都睡不着，很多事情，想了有想，最终
做了以下几件决定：

  1. 删除微博，遗憾的是不能注销微博
  2. 注销豆瓣
  3. 注销QQ空间
  4. 删除一切微信信息

这些社交圈耗费我太多的时光，也消耗了太多的注意力

三年前，一位玩的很要好的女生，在我认识她之前，有了男朋友，但我们俩聊得很嗨
一个月后，我发现，她的言行举止正在慢慢影响我的老婆观…想了想不对劲，加上本身公司也感觉没前途，于是离职了

现在，忽然间发现，又陷入了三年前的困境了…可是我刚入职…接下来要何去何从

对于已婚人士，在一起吃饭除了聊孩子就是聊老公，从某些方面说，给了我动力，我也从来没那些想法。

但是，对于未婚女生，这个诱惑力还是太大了…瞬间感觉以后找的老婆就要像她一样…然后对现任女朋友各种不爽…

每天，我都约束自己不要想太多，能不和她聊天就不和她聊天，可是，有时候真的控制不住，尤其是她某个话题过来，一聊就聊开了

然后每次下班后，各种后悔中…

然后第二天上班，有走上了重复的道路

我是不是应该离职…
